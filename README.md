# 🎨 Fatema Arbabzadeh Portfolio

A modern and responsive portfolio website built for **Fatema Arbabzadeh**, showcasing her graphic design work, skills, and contact information. The website is designed with a clean and elegant interface to highlight creative projects while providing a seamless user experience across all devices.

## ✨ Features

* 🎨 Modern and minimal UI
* 📱 Fully responsive design
* 🌙 Clean and accessible layout
* 🖼️ Portfolio gallery to showcase graphic design projects
* 👩 About section introducing the designer
* ⚡ Fast performance with Vite
* 🎯 Smooth navigation and user-friendly experience

## 🛠️ Tech Stack

* **React**
* **Vite**
* **Tailwind CSS**
* **React Icons**

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Baharah-Rostami/FatemaArbabzadeh.git
```

### Navigate to the project

```bash
cd fatema-arbabzadeh
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
